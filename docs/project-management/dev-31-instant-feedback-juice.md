# DEV-31 — Мгновенный фидбэк (juice)

## Что есть сейчас
- На попадание игрока уже были базовые реакции: tint/tween у игрока и врага, knockback врага, короткий screen shake.
- На получение урона игроком были звук, tint и screen shake.
- На убийство врага был только базовый звук `enemy-down` и удаление спрайта.

## Что должно быть в хорошем продукте
- Для каждого hit-события должна срабатывать связка: **визуал + звук + изменение состояния**.
- Попадание по врагу должно давать короткий всплеск частиц/искр и явный hit-react.
- Добивание должно иметь отдельный «критический» акцент: дополнительный flash/shake и более резкий вариант SFX.
- Получение урона игроком должно сопровождаться не только shake/tint, но и локальным burst-эффектом.
- Контакт врага с игроком должен вызывать micro-stun/реакцию, чтобы столкновение ощущалось физически.

## Реализация в коде
- Добавлен hit-burst при попадании по врагу (радиальные искры через tween).
- Добавлен finisher feedback при добивании: flash, усиленный shake и питченный вариант `enemy-down`.
- Добавлен damage-burst при получении урона игроком.
- Добавлен короткий pushback игрока и micro-stun врага при контакте.

## Как проверить
1. Запустить уровень (`npm run dev`), войти в бой.
2. Ударить врага: увидеть всплеск частиц, shake, реакцию врага и обновление боевого текста.
3. Добить врага: увидеть усиленный flash/shake и услышать акцентированный kill-звук.
4. Получить урон: увидеть burst вокруг игрока, shake и HP-изменение на HUD.
